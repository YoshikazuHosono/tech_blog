# zenn

---

```text
  👇  新しい記事を作成する
  $ zenn new:article

  👇  新しい本を作成する
  $ zenn new:book

  👇  投稿をプレビューする
  $ zenn preview

  update
  npm install zenn-cli@latest


npx zenn new:article --slug 記事のスラッグ --title タイトル --type idea --emoji ✨

npx zenn new:article --slug flutter_audio_crush --title "【Flutter】audioplayersを使用して効果音を高速連続再生するとクラッシュする" --type tech --emoji ✨

npx zenn preview
```
